# 프로그래밍 기초

## 변수와 자료형

프로그래밍에서 가장 기본적인 개념은 변수입니다. 변수는 데이터를 저장하는 공간이며, 자료형에 따라 저장할 수 있는 데이터의 종류가 결정됩니다.

### 기본 자료형

- **정수 (Integer)**: 소수점이 없는 숫자 (예: 1, 42, -7)
- **실수 (Float)**: 소수점이 있는 숫자 (예: 3.14, -0.5)
- **문자열 (String)**: 텍스트 데이터 (예: "Hello World")
- **불리언 (Boolean)**: 참/거짓 값 (True, False)

## 조건문

조건문은 프로그램의 흐름을 제어하는 핵심 구조입니다.

`if score >= 90: print("A학점")elif score >= 80: print("B학점")else: print("C학점 이하")`

## 반복문

같은 작업을 여러 번 반복해야 할 때 사용합니다.

`for i in range(5): print(f"반복 {i}번째")`

## 함수

코드의 재사용성을 높이기 위해 함수를 정의합니다.

`def greet(name): return f"안녕하세요, {name}님!"`
