# Welcome!

## Overview

**CueNote** is an open-source, privacy-first desktop application for markdown note-taking with powerful AI assistance. Unlike cloud-based note apps, CueNote keeps all your data local while providing intelligent features powered by both local LLMs and cloud APIs.
