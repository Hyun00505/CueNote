# 웹 개발 입문

## HTML 기초

HTML은 웹 페이지의 구조를 정의하는 마크업 언어입니다.

# `안녕하세요!`

`웹 개발의 세계에 오신 것을 환영합니다.`

### 주요 태그

-

- : 인라인 컨테이너
- : 링크
- : 이미지

-

## CSS 스타일링

CSS로 웹 페이지를 예쁘게 꾸밉니다.

`.container { max-width: 1200px; margin: 0 auto; padding: 20px;}h1 { color: #333; font-size: 2rem;}`

## JavaScript 동적 기능

사용자 인터랙션을 처리합니다.

`document.getElementById('btn').addEventListener('click', () => { alert('버튼이 클릭되었습니다!');});`

## 반응형 웹 디자인

모바일과 데스크톱 모두에서 잘 보이도록 만듭니다.

`@media (max-width: 768px) { .container { padding: 10px; }}`
