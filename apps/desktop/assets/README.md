# Desktop App Assets

이 폴더에 앱 아이콘을 추가해주세요.

## 필요한 파일

### Windows
- `icon.ico` - Windows 앱 아이콘 (256x256 권장, 다양한 크기 포함)

### macOS (향후 지원)
- `icon.icns` - macOS 앱 아이콘

### Linux (향후 지원)  
- `icon.png` - Linux 앱 아이콘 (512x512 권장)

## 아이콘 생성 방법

1. 1024x1024 PNG 이미지 준비
2. 온라인 변환 도구 사용:
   - https://icoconvert.com/ (PNG → ICO)
   - https://cloudconvert.com/png-to-icns (PNG → ICNS)

## 참고

아이콘이 없어도 빌드는 가능하지만, 기본 Electron 아이콘이 사용됩니다.
